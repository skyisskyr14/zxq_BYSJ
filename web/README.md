# pet-boarding-web

三端统一前端演示项目（Vue2 + VueRouter + Vuex + Element-UI）。

## 启动
```
npm install
npm run dev
```
默认访问 `/login`。

## 账号
- 用户：`user1` / `123456`
- 商家：`merchant1` / `123456`
- 管理员：`admin` / `123456`

也可以在登录页使用“一键登录”快速切换身份。

## 演示流程
1. 用户端：门店列表 -> 创建预约 -> 确认订单 -> 支付成功 -> 订单详情
2. 商家端：预约管理确认 -> 入住办理分配笼位 -> 发布寄养动态 -> 退房完成
3. 退款流程：用户订单详情发起退款 -> 商家订单详情填写意见 -> 管理端退款审核通过/驳回
4. 改期流程：用户订单详情发起改期 -> 商家订单详情填写意见 -> 管理端改期审核通过/驳回

## 说明
- 所有数据来自 `src/mock` 的内存数据，刷新页面后会重置。
- 所有 CRUD 操作均在前端内存数据中生效。
